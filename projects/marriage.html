Smart Marriage System

JavaFX, JDK 1.8, MySQL | Personal Project

Associated with: University Project

Project Overview

A comprehensive marriage record management system developed with JavaFX that provides complete marital status verification and registration services. This unique solution serves as a digital ledger for marriage records with advanced search and verification capabilities.

Key Features

  • Complete marriage data management system
  • Marital status verification checker
  • Marriage registration validation
  • Advanced database search functionality
  • Secure record keeping and retrieval
  • User-friendly JavaFX interface
  • Comprehensive reporting system
JavaFX JDK 1.8 MySQL Scene Builder JDBC

System Requirements

Recommended: JDK 1.8 or compatible version
