Online Auction System

Node.js, Express.js, WebSocket, MongoDB | April 2023

Associated with: Personal

Project Overview

A real-time auction platform featuring customer, seller, and admin modules with WebSocket technology for live bidding. The system includes comprehensive security measures and user management features to ensure a safe and engaging auction experience.

Key Features

  • Real-time bidding with WebSocket technology
  • Three-tier user system (customer, seller, admin)
  • Secure JWT authentication system
  • Email notifications for bidding activity
  • User verification process
  • Comprehensive product management
  • Fraud prevention mechanisms
