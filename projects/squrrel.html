Squirrel Football Academy

Django, Stripe, Wagtail CMS, PostgreSQL, JavaScript | January 2024

Associated with: Marvelous Technologies

Project Overview

A comprehensive event management platform for Squirrel Football Academy featuring content management, event registration, and secure payment processing. The system provides a seamless experience for both administrators and participants.

Key Features

  • Dynamic event publishing and management system
  • Online registration with secure payment processing via Stripe
  • Custom-built with Wagtail CMS for easy content management
  • PostgreSQL database for reliable data storage
  • Responsive frontend with interactive JavaScript components
  • Admin dashboard for event and participant management
