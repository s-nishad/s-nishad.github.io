AI-Powered Recruitment Platform

DRF, PostgreSQL, Celery, Redis, OpenAI, Stripe, Telethon | May 2025

Project Overview

This platform leverages AI and modern backend technologies to automate and optimize recruitment processes. It integrates AI-powered resume parsing, job matching, and real-time data workflows to significantly reduce manual recruiter effort while ensuring high accuracy.

Key Features

  • Backend architecture with Django REST Framework & PostgreSQL
  • Authentication via JWT, Google, LinkedIn, and Telegram OAuth
  • AI-powered matching engine using OpenAI, LangChain, and Sentence-Transformers
  • Automated job extraction from LinkedIn & Telegram APIs
  • Asynchronous workflows with Celery & Redis for high concurrency
  • Role-based access control for different user types
